# Brand-Locked Design System Prototype Kit

A fillable trial kit for testing whether AI-generated product screens obey a brand/design system instead of producing generic SaaS UI.

## Why this exists
Inspired by: https://x.com/claudeai/status/2067325887909884315

## What it does
- Provides a first-trial packet for one real workflow.
- Includes an infographic explaining the operating model.
- Keeps a draft skill separate until evidence proves reuse.

## Quickstart
Open `prototype/Brand Lock Trial Packet.md`, fill it during a real task, and attach proof before changing claims.

## Latest status
Template-ready, not validated. Latest improvement: first-trial packet and GitHub mirror created on 2026-06-19.
